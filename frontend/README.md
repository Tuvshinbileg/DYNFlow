## CRUD Demo Page

A complete CRUD (Create, Read, Update, Delete) demo page is available at `/crud` that showcases all installed components:

- **Table**: Display items in a formatted table with actions
- **Dialog**: Modal for creating and editing items
- **Form Components**: Input, Select, Textarea, Label for data entry
- **Calendar/Popover**: Date picker for selecting creation date
- **Switch**: Toggle notification settings
- **Checkbox**: Display notification status
- **Badge**: Show status and priority with color-coded badges
- **Card**: Container for layout and summary cards
- **Button**: Action buttons for CRUD operations
- **Sonner**: Toast notifications for user feedback

Visit `http://localhost:3000/crud` to see the CRUD demo in action!
